## 프로그램 전반적인 목적
생활 환경에 대한 공공데이터와 웨어러블 기기로부터 추출한 건강 데이터를 고려하여 사용자의 위험 상태를 분류한다. 웨어러블 기기로부터 불러온 건강 데이터(심박수, 행동 인식 데이터)를 LSTM을 이용해 미래의 건강 상태를 예측한다. 다중 선형 회귀 방식을 통해 공공 데이터를 위험 상태와 비위험 상태로 분류한다.
